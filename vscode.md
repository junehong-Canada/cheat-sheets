|---|---|---|
| Quick Open (Find File) | Ctrl+P | Cmd+P |
| Delete Line | Ctrl+Shift+K | Cmd+Shift+K |
| Multi-select Cursor | Ctrl+D | Cmd+D |
| Copy Line | Ctrl+Alt+D | Cmd+Shift+D |
| Duplicate Line | Ctrl+Shift+L | Cmd+Shift+L |
| Copy & Paste Line | Ctrl+Alt+(downarrow) | Cmd+Alt+(downarrow) |
